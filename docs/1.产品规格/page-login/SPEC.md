# 登录页规格文档

## 1. 概述

- **功能描述**: 处理用户微信登录和手机号授权
- **业务背景**: 需要获取用户信息和手机号才能使用完整功能
- **优先级**: P0

## 2. 页面结构

| 页面 | 路由 | 说明 |
|-----|------|-----|
| 登录页 | `/pages/login/login` | 非 TabBar 页面 |

## 3. 功能需求

### 3.1 核心功能

| 功能点 | 描述 | 验收标准 |
|-------|------|---------|
| 微信登录 | 获取用户微信信息 | 获取头像和昵称 |
| 手机号授权 | 绑定用户手机号 | 获取加密手机号 |
| 登录状态检查 | 检查是否已登录 | 已登录跳转首页 |
| 自动登录 | 启动时自动检查登录态 | 自动跳转到对应页面 |

### 3.2 用户交互流程

```
进入登录页 → 检查登录状态
                  ↓
        已登录 → 跳转到首页
                  ↓ 未登录
         显示登录按钮
                  ↓
         用户点击登录
                  ↓
         微信授权获取用户信息
                  ↓
         显示手机号绑定按钮
                  ↓
         用户点击绑定手机号
                  ↓
         保存用户信息 → 跳转首页
```

## 4. 数据模型

### 4.1 全局状态

```typescript
// app.ts globalData
interface GlobalData {
  userInfo: any;              // 用户信息
  hasUserInfo: boolean;       // 是否已获取用户信息
  phoneNumber: string;        // 手机号
  hasPhoneNumber: boolean;    // 是否已绑定手机号
  accessToken: string | null;
  refreshToken: string | null;
  openid: string | null;
  sessionkey: string | null;
  code: string | null;
}
```

### 4.2 API 接口

| 接口 | 方法 | 描述 |
|-----|------|-----|
| `/api/users/openid` | POST | 通过 code 获取 openid |
