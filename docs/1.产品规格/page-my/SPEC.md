# 我的页面规格文档

## 1. 概述

- **功能描述**: 个人中心页面，展示用户信息和功能入口
- **业务背景**: 用户查看个人信息、管理订单和设置
- **优先级**: P0

## 2. 页面结构

| 页面 | 路由 | 说明 |
|-----|------|-----|
| 我的 | `/pages/my/my` | TabBar 页面 |

## 3. 功能需求

### 3.1 核心功能

| 功能点 | 描述 | 验收标准 |
|-------|------|---------|
| 用户卡片 | 显示用户头像、昵称等 | 展示基本信息 |
| 订单入口 | 查看不同状态的订单 | 跳转订单列表 |
| 功能列表 | 其他功能入口 | 展示设置、帮助等 |

### 3.2 页面布局

```
┌────────────────────────┐
│      顶部 TabBar        │
├────────────────────────┤
│                        │
│   ┌────────────────┐   │
│   │  用户中心卡片   │   │  ← user-center-card 组件
│   │  头像 昵称     │   │
│   └────────────────┘   │
│                        │
│   ┌────────────────┐   │
│   │  订单分组       │   │  ← order-group 组件
│   │  待付款/待发货... │  │
│   └────────────────┘   │
│                        │
│   ┌────────────────┐   │
│   │  功能列表       │   │
│   │  - 设置        │   │
│   │  - 帮助        │   │
│   │  - 关于        │   │
│   └────────────────┘   │
│                        │
└────────────────────────┘
```

## 4. 组件使用

| 组件 | 用途 | 来源 |
|-----|------|-----|
| user-center-card | 用户信息卡片 | 页面私有组件 |
| order-group | 订单分组入口 | 页面私有组件 |

- [ ] 订单分组显示正确数量
