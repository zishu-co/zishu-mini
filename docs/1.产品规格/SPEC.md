# 产品规格总览

> 本文档概述小程序的所有功能和页面规划。

## 📱 页面清单

### TabBar 页面

| 页面 | 路由 | 描述 | 优先级 |
|-----|------|-----|-------|
| 学习/首页 | `/pages/index/index` | 展示学习内容、试卷列表 | P0 |
| 创新 | `/pages/inno/index` | 创新内容展示与分享 | P1 |
| 目标 | `/pages/aim/index` | 目标管理与跟踪 | P1 |
| 活动 | `/pages/event/index` | 活动列表与参与 | P1 |
| 我的 | `/pages/my/index` | 个人中心与设置 | P0 |

### 非 TabBar 页面

| 页面 | 路由 | 描述 | 优先级 |
|-----|------|-----|-------|
| 登录 | `/pages/login/login` | 用户登录与授权 | P0 |
| 日志 | `/pages/logs/logs` | 操作日志查看 | P2 |
| 测试 | `/pages/test/test` | 功能测试页面 | P2 |
